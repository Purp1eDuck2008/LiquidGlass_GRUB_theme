LiquidGlass GRUB Theme

LiquidGlass is a sleek and minimalistic GRUB theme inspired by modern glassmorphic design. It emphasizes clean visuals, smooth animations, and transparent elements, making your boot menu both stylish and user-friendly.

Features

Elegant glassmorphic design

Light and dark theme support

Easy installation and customization
